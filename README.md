# 雨雲列伝

# ゲームの説明
## ゲームの概要

* 雨雲から雨を発射し、花のつぼみに当てて花を咲かせましょう 
* 1分間にできるだけ多くの花を咲かせることを目指すゲームです

## 操作方法

* Spaceキー・・・雨を発射
* 左右カーソル or A,Dキー・・・雨雲を移動させる

## 詳細なルール

* 地上に流れてくるつぼみに雨を当てることで得点が加算されます
* 誤って猫に当ててしまうと減点です

* 雨を発射しすぎると雨雲が小さくなり雨を発射することが出来なくなります
* 雨雲が地上から湧く水蒸気に触れることで雨の発射数が増えます

* ただし、雲が大きくなりすぎると移動速度が遅くなってしまうので注意が必要です
